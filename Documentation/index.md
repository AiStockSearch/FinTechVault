Обновленное техническое задание для торгового терминала Wealth Project
1. Цель проекта (по PMI/PMBOK)
1.1. Миссия проекта
Миссия: Создать инновационный торговый терминал Wealth Project, объединяющий торговлю на централизованных (CEX) и децентрализованных (DeFi) биржах, управление портфелем, моделирование стратегий с использованием языка Julia и публикацию монетизируемого контента на блокчейне. Терминал упрощает доступ к сложным финансовым инструментам, предоставляя единый интерфейс для торговли, аналитики, монетизации идей и гибкого управления активами с автоматическим распределением прибыли.
Цель:

Разработать кроссплатформенный терминал (десктоп на Wails, мобильный на React Native) для торговли RVA-продуктами, NFT, криптоактивами и облигациями.
Интегрировать API бирж (Bybit, OKX, Bitfinex) и DeFi-протоколы (Uniswap, Aave, Curve).
Реализовать создание пользовательских криптоиндексов и моделирование стратегий через Julia-сервер.
Создать систему публикации инвестиционных статей на блокчейне с транскрибацией и переводом.
Поддержать управление портфелем с автоматическим распределением прибыли (например, 20% на вывод, страховку, долгосрочный, среднесрочный, кэшфлоу-продукты).
Обеспечить безопасность через NFC-подтверждения, шифрование и двухфакторную аутентификацию (2FA).

Ключевая ценность:

Единый интерфейс для краткосрочных, среднесрочных и долгосрочных портфелей.
Автоматизация распределения прибыли по заданным процентам.
Монетизация инвестиционных идей через статьи.
Прямое управление и моделирование для трейдеров всех уровней.

2. Пример использования Wealth Project
2.1. Сценарий: Управление портфелем с целевым доходом 20% годовых
Контекст: Розничный трейдер планирует 20% годовых (~1.67% в месяц). Формирует месячный watch-лист, управляет сделками, распределяет прибыль по 20% в пять направлений, отслеживает краткосрочный, среднесрочный и долгосрочный портфели, а также индекс в едином интерфейсе.
2.1.1. Формирование watch-листа

Цель: Создать watch-лист на месяц для 1.67% доходности.
Действия:
Открывает модуль "Watch-лист".
Добавляет активы через API (Bybit, OKX, Bitfinex) и DeFi:
Криптовалюты: BTC, ETH, SOL.
NFT: Ethereum-коллекции (Bored Ape Yacht Club).
Облигации: токенизированные через Aave.
RVA: токены, привязанные к VIX.


Делит watch-лист:
Первые 2 недели: SOL, NFT (волатильные).
Вторые 2 недели: BTC, облигации (стабильные).


Устанавливает триггеры:
SOL = $200 → лимитная заявка.
ETH -5% → уведомление.
NFT-объем +20% → сигнал.


Синхронизация с Julia-сервером:
Анализ данных, прогноз доходности.
Рекомендация: "30% SOL в первые 2 недели".





2.1.2. Управление сделками

Цель: Выполнить сделки, распределить прибыль.
Действия:
Открывает модуль "Торговля".
Покупает SOL по $200 через Uniswap.
Продает по $240 через 10 дней, прибыль $200.
Подтверждает через NFC.
Распределяет $200 по 20%:
$40: Вывод на счет (NFC).
$40: Страховка (USDT).
$40: Долгосрочный (VIX/TLT).
$40: Среднесрочный (новые возможности).
$40: Кэшфлоу (акции/облигации).





2.1.3. Управление портфелями

Цель: Разделить активы на три портфеля и индекс.
Действия:
Открывает модуль "Портфель".
Видит портфели:
Краткосрочный:
Активы: SOL, NFT.
Цель: 1–2 недели.
Пример: $1000 SOL, $500 NFT.


Среднесрочный:
Активы: ETH, облигации.
Цель: 1–6 месяцев.
Пример: $2000 ETH, $1000 облигации.


Долгосрочный:
Активы: BTC, USDT, VIX/TLT.
Цель: 6+ месяцев.
Пример: $3000 BTC, $1000 USDT.




Создает "Wealth Index":
40% BTC, 30% ETH, 20% SOL, 10% облигации.
Торгует индексом.


Распределяет прибыль:
Краткосрочный: 20% в каждое направление.
Среднесрочный: 50% в индекс, 50% в долгосрочный.
Долгосрочный: аккумуляция для VIX/TLT.





2.1.4. Мониторинг и аналитика

Цель: Отслеживать прогресс, корректировать.
Действия:
Дашборд:
Доходность: +1.2% за месяц.
Графики: портфели, индекс, активы.
Уведомления: "SOL = $250, зафиксировать?".


Julia-моделирование:
Сценарий: "SOL -10% → 20% в BTC".
Рекомендации в интерфейсе.


Публикация статьи:
"20% годовых через Wealth Index".
Голос → транскрибация → перевод.
Монетизация: $5/месяц, инвестиции.





2.1.5. Итог

Единый интерфейс:
Watch-лист с триггерами.
Прибыль: 20% в вывод, страховку, долгосрочный, среднесрочный, кэшфлоу.
Портфели и индекс.
Аналитика, Julia, публикация.


Доходность: +1.2% за месяц, прогресс к 20% годовых.
